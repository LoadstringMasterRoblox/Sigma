Get started by following one of our iOS quick start guides

1. Follow the iOS guide to configure your project and create the codemagic.yaml file.
2. Save the codemagic.yaml file to the project root folder.
3. Push the added file to the repository. From command line:

```
git add codemagic.yaml
git commit -m 'Add first workflow'
git push
```
